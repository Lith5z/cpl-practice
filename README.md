# cpl-practice

个人的cpl和git练手项目
目前只有一个矩阵计算器和一个线性方程组计算器(solver)

## matrix 矩阵计算器

某次OJ作业写的，用于给线性方程组计算器创造样例

## solver 线性方程组计算器

精度为0.0001
源文件使用GBK编码

实现以下 4 个功能：
1. 利用克莱姆法则求解
2. 利用高斯消元法求解 (bug: 未通过所有情况的测试)
3. 求矩阵的行列式
4. 将矩阵化为行最简型，并求矩阵的秩

提供一个监视模式，显示每一步计算的过程 (bug: 对高斯消元法不生效)
